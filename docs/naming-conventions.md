# Naming Conventions

## Principles

### Packages: short, meaningful, no underscore, no hyphen (e.g., `auth`, `config`).

* ✔️ `authsvc`
* ✔️ `authservice`
* ❌ `AuthService`
* ❌ `auth-service`

### Files: lowercase, underscore, no hyphen.

* ✔️ `user_service.go`
* ❌ `AuthService.go`
* ❌ `auth-service.go`
