# Go QuickStart

## Install Go
* Check installation:
```
go version
```

## Create a Project Directory
```
mkdir myproject
cd myproject
```

## Initialize a Go Module
```
go mod init github.com/yourname/myproject
```

## Run Your Project
```
go run main.go
```

## Build a Binary
```
Build a Binary
```

## Add Dependencies
```
go mod tidy
```
